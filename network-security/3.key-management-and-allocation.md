# 密钥管理与分配

## 密钥生命期
- 产生/等级
- 分配
- 使用/启用/停用
- 更新/替换
- 撤销
- 销毁

## 密钥分级
- 初级密钥
    - 加密/解密实际数据。
    - 分类：
        - 初级文件密钥：生命周期与所保护的文件生命周期一样长。
        - 初级通信密钥：生命周期很短，一次密钥使用一次或多次。
    - 由二级密钥加密后传输。
- 二级密钥
    - 保护初级密钥。
    - 由主密钥加密后通过互联网自动传输。
- 主密钥
    - 保护二级密钥。
    - 人工分配，由专职密钥分配人员分配并由专职人员妥善安装。
